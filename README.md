# Predictive Maintenance & RUL Prediction

This project applies machine learning techniques to detect anomalies and estimate remaining useful life (RUL) in industrial systems using NASA C-MAPSS dataset.

## Key Features
- Anomaly detection using Isolation Forest
- RUL (Remaining Useful Life) estimation
- Time-series sensor analysis
- Visualization of degradation patterns

## Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## Results
- Successfully detected degradation patterns
- Observed correlation between anomaly score and RUL
- Demonstrated predictive maintenance capability

## Dataset

The dataset used in this project is the NASA C-MAPSS Turbofan Engine Dataset.

It can be downloaded from:
https://data.nasa.gov/

Specifically, the FD001 subset was used in this project.

## How to Run

1. Download the dataset (FD001) from NASA
2. Place the files inside a `data/` folder:
   - train_FD001.txt
3. Run the notebook
